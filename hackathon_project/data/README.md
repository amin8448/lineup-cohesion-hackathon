# Data Directory

This directory stores generated analysis outputs.

Files are generated by running the analysis scripts:
- cohesion_results.csv (from 03_full_analysis.py)
- team_season_summary.csv (from 03_full_analysis.py)
- cohesion_results_optimized.csv (from 05_revalidate_optimized.py)

Raw match data is loaded dynamically via kloppy from IMPECT Open Data.
